# Profile Management System

Full-stack user profile management system with Next.js 14 frontend and Express/Prisma backend.

## Features

- User profile creation and editing
- Avatar upload with image processing
- Privacy settings and profile visibility control
- Profile settings management
- JWT authentication

## Tech Stack

**Frontend:**
- Next.js 14
- TypeScript
- Tailwind CSS
- React Query
- Axios

**Backend:**
- Node.js + Express
- Prisma ORM
- PostgreSQL
- JWT authentication
- Multer + Sharp for image processing

## Development

```bash
# Install dependencies
npm install

# Run backend
cd backend
npm run dev

# Run frontend
cd frontend
npm run dev
```
